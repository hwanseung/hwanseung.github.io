논리학

*왜 배우는가?

-새로운 지식을 탐구할 때 필요한 논리적 도구

-조리있고 설득력이LT는 글을 쓰거나 말할 수 있음.

 

*개념

- 물의 특유한 속성을 반영하는 사고형식

- 개념은 단어로만 표현가능

but 모든 단어 !=> 개념표현

ex)접속부사,조사와 같은 허사

- 동일한 개념이 다른 단어에 의해 표현될 수 있음

ex) 의복,옷 교사,선생

- 동일한 단어이지만 다른 개념표현가능

ex) 눈이 멀다 , 거리가 멀다.

 

*개념의 내포

개념이 반영하고 있는 사물의 특유한 속성

ex) 동물(개념) -> 유기체로서 샐명을 가지고 스스로 이동하는 생명체의 총체

 

*개념의 외연

개념이 반영하고 있는 사물의 총체

ex) 동물(개념) -> 인간,소,돼지,말, 뱀, 개구리.

 

*개념 정의방법

정의되는 개념 = 종차 + 최근 류개념

 

종차=> 어떤 사물을 특정짓는 본질을 말함.

ex) 대학생을 정의

유개념 => 학생 (대학생을 포함하는 범위)

종개념 => 초등학생,중학생,고등학생 (유개념에 포함된 하위개념)

 

-정의하는 개념의 외연은 정의되는 개념의 외연과 완전히 동일해야 한다.

=>정의의 범위가 너무 넓거나 좁아서는 안된다.

-정의하는 개념에 직접 혹은 간접적으로 정의 되는 개념이 포함되서느 s안된다.

ex)독재자에대한 정리 => 독재를 실시하는사람 (독재는 머임?)

-정의하는 개념은 일반적으로 부정개념을 포괄시키면 안됨.

ex)놀부는 흥부가 아니다. 예외) 대머리(머리카락없다.),고아(부모가 없다.)

-모호한 어구 포함안됨.

 

*동일개념

내포는 서로 차이가 있지만 외연은 완전히 합치되는 두 개념 관계

ex)서울,한국의 수도

 

 

*대소관계

한 개념의 외연이 다른 개념의 외연 속에 완전히 포함

ex) 학생,중학생 유개념,종개념

 

*교차관계

두 개념의 외연이 일부분 서로 합치되는 두 개념간의 관계

ex) 학생,여자 (어떤 학생은 여자고 어떤 여자는 학생일 수 있음.)

 

*모순관계

하나의 가까운 유개념 속에 속하는 두 종개념의 외연히 완전히 다르며, 그들의 외연을 합한것이 그 유개념의 외연과 같을 때

ex)남자.여자 => 유개념은 인간 유개념의 종개념은 남자,여자 둘 밖에 존재 안함.

 

*반대관계

하나의 유개념에 종속되는 두 종개념의 외연이 완전히 다르며, 그 외연을 합한것이 그 유개념의 외연보다 좁은 두 종개념간의 관계를 반대관계

ex)흰색,검정색 =>유개념은 색상이며 그 유개념엔 흰색,검정만 있는건 아님. 다른거 많음.

 

*판단이란?

사고의 대상에 대하여 그 무엇을 단정하는 사고의 형식 => 긍정혹은 부정으로 표시;;

판단의 진리성은 주관적이 아닌 객관적으로 판단해야함.

 

*타당한 판단을할려면?

1.사실에 근거

2.논리에 맞음

3.실천의 필요에 부응.

ex) Q)새벽조깅이 여학생이 좋나여 나쁘나요?

A)생리학적 원리에 부합하면 좋고 아님 나쁘고 =>그걸 몰라서 묻는 소리임

4.판단은 언어로 통해 표현되므로 언어를 잘 표현해야함.

 

*판단과 문장이 다른점

1.다른 문장이 동일한 판단을 표현 할 수 있음.

ex)모든 사물은 변한다.,변하지않은 사물은 없다.

2.동일한 문장이 다른 판단을 할 수있 음.

ex)그는 이제야 눈을 떳다. (그는 지금까지 감았던 눈을 떳다, 그는 이제야 사물을 제대로 보게 되었다.)

3.어떤 판단인등 모두 문장에 의해 표현되지만 모든 문장이 다 판단을 표현하는 것은 아님.

개연판단 -> 지금으로써는 참이나 언제 바뀔 수 있다.

필연판단 -> 참이나 거짓을 판단할수 있으나 새로운 정보 없음.

 

A판단 전칭긍정판단 모든 S는 P이다.

E판단 전칭부정판단 모든 S는 P가 아니다.

I판단 특칭긍정판단 어떤 S는 P이다.

O판단 특칭부정판단 어떤 S는 P가 아니다.

 

*A판단과 E판단 (반대관계- 두 판단이 동시에 참일수 없으나 거짓일수는 있는 관계)

A판단 참 => E판단 거짓

A판단 거짓 => E판단 미정

E판단 참 => A판단 거짓

E판단 거짓 => A판단 미정

 

*I판단과 E판단 (모순관계-두 판단 모두 참이거나 거짓일수 없음.)

E판단 참 => I판단 거짓

I판단 참 => E판단 참

 

*A판단과 I판단 (대소관계 - 두 판단이 동시에 참이 되가나 거짓이 될수 있는 관계)

A판단 참 => I판단 참

A판단 거짓 =>I판단 미정

I판단 참 => A판단 미정

I판단 거짓 =>A판단 거짓

 

 

*I판단과 O판단 (소반대관계- 두 판단이 모두 참일 수 있으나 모두 거짓일 수 없는 관계)

I판단 참 => O판단 미정

I판단 거짓 =>O판단 참

O판단 참 => I판단 미정

O판단 거짓 => I판단 참

 

*주연

어떤 판단에 있어서 주장하는 바가 주어 또는 술어로 되어 있는 개념의 외연에 전부에 미치는 경우.

전칭판단의 주어와 부정판단의 술어 부분은 주연임.

 

*부주연

어떤 판단에 있어서 주장하는 바가 주어 또는 술어로 되어 있는 개념의 외연에 일부에 미치는 경우.

특칭판단의 주어와 긍정판단의 술어 부분은 부주연임.

 

*환질법 : 원래의 질을 바꾸는 방법 즉 긍정 판단을 부정판단 Or 부정판단을 긍정판단.

1.전제의 질을 바꿔야하며, 전제의 술어를 부정개념으로 고쳐야함.

 

*환위법 : 주어와 술어의 위치를 바꾸는 법

1.전제 판단의 질을 바꾸지 말아야 되며

2.전제중에 부주연되었던 개념은 환위후 주연이 되지 말아야 한다.

*환질환위법 => 환질 후 환위.
