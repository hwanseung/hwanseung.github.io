Python으로 구현된 Lex,yacc 모듈인  PLY를 설치해봅시다.

일단 PLY 공식사이트는  http://www.dabeaz.com/ply/ 입니다.ㅋ

 


이 사이트의 하단에 보면 다운로드가 있습니다.

다운 받아서 적당한 곳에 둡니다.
그리고는 압축을 풉니다.

압축을 풀고 해당 폴더에 들어가면 setup.py 파일이 존재하구요 ㅋ

$ sudo python3 setup.py install
명령어를 치시면 아래처럼 설치가 되는걸 볼수 있습니다.
저는 python3에 깔기 위해서 python3라고 명시했는데 아니신분들은 python이라고 쳐도 무방합니다.
