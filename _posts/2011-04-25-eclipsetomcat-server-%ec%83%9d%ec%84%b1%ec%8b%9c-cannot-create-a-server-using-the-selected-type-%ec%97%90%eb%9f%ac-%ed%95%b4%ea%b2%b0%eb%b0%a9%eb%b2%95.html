이클립스에서 종종 프로젝트의 경로를 바꾸어서 하다가

서버를 새로 연결 해야 될 경우 지웠다가 다시 서버를 만들때 보면

 Cannot create a server using the selected type 에러를 볼 수 있는데

이럴때는

workspace의  

.metadata\.plugins\org.eclipse.core.runtime\.settings에 보면

org.eclipse.wst.server.core.prefs

org.eclipse.jst.server.tomcat.core.prefs
두개의 파일이 있는데 이것을 삭제하고 다시 이클립스를 시작하면

잘 생성이 된다.
